/createUser -> creates an user and returns it

/user/<int:user_id> -> get/ put(update)/ delete one user

/usersVideos/<int:user_id> -> returns all videos for one user

/video/<int:video_id> -> get/ put(update)/ delete one video

/badges/<int:badge_id> -> put(update)/ delete one badge

/userBadges/<int:user_id> -> calculates and return badges for one user
